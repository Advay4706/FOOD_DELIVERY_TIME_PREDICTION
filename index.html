How It Works

Two regression models are trained and compared on an 80/20 train-test split with R² and RMSE as evaluation metrics.

📈

Linear Regression

Baseline model. Assumes a linear relationship between all input features and delivery time. Fast to train and interpretable.

Baseline
🌲

Random Forest

Ensemble of 100 decision trees. Captures complex non-linear interactions. Significantly outperforms Linear Regression on this dataset.

Best Model
📐

Haversine Distance

Distance is computed from GPS coordinates using the Haversine formula — the shortest path over Earth's curved surface.

Feature Engineering
🏷️

Label Encoding

Categorical features (weather, traffic, vehicle type, city) are encoded into integers using scikit-learn's LabelEncoder.
